# Community Catalyst Connectors

Connectors built and maintained by the community.

## Contributing

To contribute a connector:

1. Fork this repository
2. Create a folder: connectors/community/[your-github-username]/[connector-name]/
3. Add your connector files:
   - [connector-name].bpmn
   - [connector-name].element.json
   - [connector-name].n8n.json
   - README.md
4. Submit a pull request
