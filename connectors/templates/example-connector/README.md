# Example Catalyst Connector

This is a placeholder example showing the structure of a Catalyst Connector.

## Files

- example-connector.bpmn - BPMN process definition
- example-connector.element.json - Element template for Camunda Modeler
- example-connector.n8n.json - n8n workflow definition

## Structure

Each Catalyst Connector must have:
1. BPMN file with process definition
2. Element template JSON with connector configuration
3. n8n workflow JSON with automation logic
4. README.md with documentation
