# Google Sheets Connector

> Read, append, and update rows in Google Sheets spreadsheets

## Overview

This connector enables integration with Google Sheets, allowing you to:
- Read data from spreadsheets
- Append new rows
- Update existing rows

## Authentication

Google OAuth credentials are configured in n8n (NOT in Camunda).
