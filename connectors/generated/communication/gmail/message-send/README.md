# Catalyst Gmail Connector - Message - Send Connector

Send a message

## Installation

### 1. Import n8n Workflow

1. Open your n8n instance
2. Click **Workflows** > **Import from File**
3. Select `gmail-send-message.n8n.json`
4. **Configure credentials** for Gmail in n8n
5. **Activate** the workflow

Webhook URL: `http://catalyst-n8n:5678/webhook/catalyst-gmail-send-message`

### 2. Deploy Element Template

Copy `gmail-send-message.element.json` to your Camunda Modeler templates directory.

## Configuration

### n8n Credentials

This connector requires Gmail credentials to be configured in n8n:

- `oAuth2`

Configure credentials in n8n under **Settings** > **Credentials** > **Add Credential**.

### Webhook URL Allowlist

Ensure your Catalyst Bridge allows the webhook URL prefix:
```
http://catalyst-n8n:5678/webhook/
```
